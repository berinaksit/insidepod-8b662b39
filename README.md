# Inside Pōd

Inside Pōd is an AI-native product insight platform built with Lovable.

It helps product teams turn scattered qualitative and quantitative data
(user feedback, research, analytics) into clear, decision-ready insights.

## How it works
- Data is connected inside Lovable
- AI synthesizes signals across sources
- Outputs summaries, insights, and suggested next steps

## Tech stack
- Lovable (no-code)
- React + TypeScript
- Supabase
- Gemini LLM

## Setup
This project runs inside Lovable.
No local setup is required to run the demo.
